# Retail Sales Forecasting – Full MLOps Pipeline  

**End-to-end production-grade daily sales forecasting system** built from real transaction data.

**Live Interactive Dashboard** → [https://huggingface.co/spaces/Rameen05/retail_sales_forecast](https://huggingface.co/spaces/Rameen05/retail_sales_forecast)  

[Demo](https://huggingface.co/spaces/Rameen05/retail_sales_forecast/resolve/main/demo.gif) 

## Features
- Raw transaction processing (Quantity × UnitPrice)
- Automatic daily aggregation
- Facebook Prophet model with yearly + weekly seasonality
- Apache Airflow orchestration (daily run at 1 AM)
- MLflow experiment tracking (ready)
- 30-day interactive forecast with confidence intervals
- Live Streamlit dashboard with metrics & visualizations
- Fully containerized & deployed on Hugging Face Spaces
